Detailed Description

Manages event subscriptions and publications for immediate, synchronous communication.

Systems can subscribe to specific event types. When an event is published, the dispatcher immediately invokes all registered callback functions for that event type. Subscriptions are managed by EventHandle objects, which automatically unsubscribe upon destruction.
