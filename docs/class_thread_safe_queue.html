Detailed Description

template<typename T>
class ThreadSafeQueue< T >

A generic, thread-safe queue for concurrent data access.

This class wraps a standard std::queue and protects it with a mutex to allow safe pushing and popping of elements from multiple threads. It uses a condition variable to provide a blocking WaitAndPop method, which efficiently waits for an item to become available without busy-waiting.
