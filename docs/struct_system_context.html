Detailed Description

A container for shared game services and managers.

This struct provides a centralized access point to all major systems and data managers in the game. An instance of this context is passed to each system, allowing them to interact with other parts of the engine without requiring a complex web of direct dependencies. This promotes loose coupling and simplifies system design.
