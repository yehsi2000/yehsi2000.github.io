Detailed Description

template<typename T>
class ComponentArray< T >

A cache-friendly container for a single type of component.

Stores components of a specific type in a contiguous array for fast iteration. It uses a map to link entity IDs to their component's index in the array, allowing for efficient addition, removal, and access of components.

Template Parameters
TThe type of component to store.
